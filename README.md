# StrongerMouse：罗技鼠标宏
基于lua脚本的鼠标自动下压，适用于PUBG的压枪辅助

内含6把枪械的下压数据，能方便地调整参数和开关功能。  
该脚本只是模拟人手下移动作，所以没办法将弹道压至一个点或者弹道分布一条水平线，只能作为降低游戏上手难度的压枪辅助来使用。

## 主要功能介绍
### G4 / G5：功能开启键。  
G4按下第一次是M4A1、第二下是AUG、第三下是UMP。  
G5分别是Vector、M762、AKM。  
G6：关闭压枪功能。  
当已经切换到UMP，想切换到M4则需要按下G6关闭，再按一次G4。 

CapsLock：对应枪械的满配、裸配切换不同下压力度。  
关闭情况下对应满配，CapsLock开启（灯亮）对应裸配。  

开启功能后，按住 Alt + 左键：站姿状态下压力度；  
按住shift+左键：屏息状态下压力度。    

### 下压模式的切换
默认触发模式（0：左键下压；1：按住右键+左键下压）  
default_trigger_mode = 0   -- 默认0，可自行修改  
选择按住右键+左键下压可以避免功能开启后，丢投掷物时也自动下压。  
但这对于有一定游戏经验的玩家来说，选择这种开枪模式要花大量时间适应。  
所以提供了切换选择，根据自身情况自行调整。

## 调试模式
DEBUG_MODE = true 时，每次移动会附加一个水平偏移（debug_horizontal_shift 像素），便于调试弹道。
